# Standard-digital-clock

A simple Digital Clock built using Python. It displays the current time in real-time with a clean and user-friendly interface.

##  Features
- Displays current time
- Real-time clock updates
- Simple and clean GUI
- Beginner-friendly project

##  Technologies Used
- Python
- Tkinter

##  Project Structure
```
Standard-digital-clock/
│── main5.py
│── README.md
```

##  How to Run
1. Install Python 3.
2. Download or clone this repository.
3. Run:

```bash
python main5.py
```
